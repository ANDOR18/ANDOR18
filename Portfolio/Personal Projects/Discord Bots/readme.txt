More info about these Discord bots:

Liguini is a humorous general purpose bot made for a personal discord server. It's likeness is based off of Luigi from Super Mario and
it's functions include:
	- Youtube music player
	- Option picker
	- Make luigi sounds
	- Append -ini to the end of a user's name
	- Scrape the web for images of pasta and pizza

Ditto is a joke bot that can copy most aspects of a user's discord profile,
including profile picture, nickname, and tags. It's likeness is based off the Pokemon of the same name.

Character Select is a bot made for the California University of PA Smash Ultimate server. It's primary function is to assign roles associated
for each player character in the game, as well as general purpose roles.
